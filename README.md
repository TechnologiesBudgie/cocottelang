# Cocotte

Cocotte is a beginner-friendly, powerful programming language designed to actually be easy, while staying as powerful as Rust, Python or C++.

## Install

```bash
curl -fsSL https://cocotte-lang.pages.dev/install.sh | sh
````

## Example

```cot
func greet(name)
    print "Hello " + name
end

greet("World")
```

## Features

* Super simple syntax
* Compiled + interpreted
* Cross-platform
* GUI support (Charlotte)
* Modules & libraries
